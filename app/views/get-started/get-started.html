NHS login is currently in a private beta, during which we will be testing the onboarding process with a small number of partners to understand the timescales involved and to refine and improve the process.

NHS login is due for public beta in March 2019.

Exploring technical integration

Technical teams can find step-by-step information on setting up the connection to NHS login on our GitHub repository.

From the GitHub they can request access to the Sandpit which is a copy of our production code environment which will help you learn how to integrate with our service and mention stub?

We also have a Slack channel where our NHS login developers are on hand to answer technical questions and support partner organisations.
