Why use NHS login?

If you’re building a health or social care app or website for patients, NHS login is a free, flexible solution which can take away the headache of creating an account for a user, verifying the users’ identity and linking it to an NHS number.

How it works

NHS login is an internet facing system for identifying and verifying a user’s identity, which uses the Open ID Connect and the Oauth2.0 protocols.

It also uses FIDO (Fast Identity Online) specifications which improve the user experience by allowing Fingerprint ID and Face ID to seamlessly access native apps.

Safety and security

Proving users’ identities using the NHS login meets the published standard for identity checking in health and care services. Read about the standards on proving identity that NHS login follows.

NHS login stores and processes users’ personal information securely and in line with strict standards. Read more about data storage.

Get started

NHS login is currently in a private beta.

Due to the number of partners waiting to use NHS login we are having to prioritise requests to join the public beta. Prioritisation will be based on a variety of factors such as strategic fit and benefits to health and social care.
